# Nested Themes

Clarity provides a built in dark theme which can be loaded and dynamically enabled
in the browser.

```scss
// import into sass or css
@import '../dist/lib/styles/theme.dark.min.css';
@import '../dist/lib/styles/theme.light.min.css';
// import instead of global.min.css;
@import '../dist/lib/styles/module.reset.min.css';
@import '../dist/lib/styles/module.typography.min.css';
@import '../dist/lib/styles/module.layout.min.css';

// or import in html
<link href="/node_modules/@cds/core/styles/theme.dark.min.css" rel="stylesheet">
<link href="/node_modules/@cds/core/styles/theme.light.min.css" rel="stylesheet">
<link href="/node_modules/@cds/core/styles/theme.light.min.css" rel="stylesheet">
<link href="/node_modules/@cds/core/styles/theme.light.min.css" rel="stylesheet">
<link href="/node_modules/@cds/core/styles/theme.light.min.css" rel="stylesheet">
```

```html
<body cds-theme="dark">
  ...
  <div cds-theme="light">
    ...
    <div cds-theme="dark">...</div>
  </div>
</body>
```
